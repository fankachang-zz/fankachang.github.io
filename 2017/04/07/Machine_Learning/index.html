Machine Learning

  • 學習來源：機器學習基石
  • 演說者：Hsuan-Tien Lin

共通性:

  • 學習是透過 觀察 經由 腦部理解 最後發展為 技巧
    • 例：經由 學習 理解 語言 而熟練後能 演說
    • 例：經由　上課　理解　內容　而熟知　課程　
  • 機器學習是透過　資料　經由　ML 最後發展為　技巧
    • 例如：讀取 歷年資料 經由 ML 後能給予　未來股市預測

差異姓：

同樣認知一張照片中有樹木的存在

  • 透過程式撰寫定義進行判定
  • 由 ML 自行定義

機器學習運用的領域：

  • 機器人送到火星會遇到的事情無法提前撰寫規則
  • 視覺辨識
  • 聲音辨識
  • 人類未想過的（十秒判斷股票是否要買賣）
  • 網路應用（服務各式各樣的使用者）

適不適合使用機器學習的三個關鍵：

  • 效能表現的增進（有潛藏的模式可以學習）：股市目標
  • 有規則但不知道怎麼寫下來
  • 具有資料

哪些領域可以運用：

  • 食：
    • Data：Twitter Data
    • Skill：告訴我們這家餐廳衛生狀況如何
  • 衣：
    • Data：銷售數字＋穿搭衣服資料
    • Skill：推薦顧客如何搭配才好看
  • 住：
    • Data：建築型態與能耗狀況
    • Skill：預設房子能源消耗狀況
  • 行：
    • Data：交通號誌圖片與意思
    • Skill：自動辨識交通號誌
  • 育：
    • Data：線上系統給學生答題
    • Skill：自動判斷學生程度能力給予相符合程度的題目
  • 樂：
    • Data：
    • Skill：推薦給客戶喜好的電影
